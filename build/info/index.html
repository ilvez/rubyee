Kuidas Rubyest toimib?

Tänu ühisele koostööle

  • Fraktal vastutab visuaalse disaini eest
  • Domify majandab ja hostib
  • Soluvi sponsoreerib ruby.ee domeeniga

Tänu tööriistade tegijatele

  • linux - ökosüsteem
  • nginx - server
  • middleman - staatiliste lehtede generaator

Kuidas lehekülge uuendada?

Kogu lehe sisu asub githubi rubyee hoidlas. Loo kloon ja salvesta täiendused ning teavita githubis uuendustest.

Lehe haldamiseks kasutame hetkel ruby Middleman generaatorit. Seega nii sisu kui ka visuaalse disaini täienduste puhul on git clone sinu sõber.

Jätame endale õiguse iga teatud aja tagant teha liikmete nimekirja puhastusi ja uuendusi ilma kirjaliku teavituseta.

Ruby.ee on avatud ka suurematale muudatustele, näiteks oma foorum jne. Ainuke tingimus, et kes käib idee välja, peab leidma jõudu ja tahet seda arendada-hallata, sest paljud meist on tööga väga hõivatud.
